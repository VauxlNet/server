# AI Agent Guidelines

## Documentation

Docs for this repo are published to https://docs.vauxl.net by Folio (https://github.com/VauxlNet/folio).

- Authored content lives in `docs/`.
- The repo root `README.md` is also included by Folio.
- Plain Markdown or MDX. No required frontmatter beyond what Folio enforces.
- Do not aggregate or copy docs from other repositories into this repo.
